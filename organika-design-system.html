Implementation Note — Gradient Border Technique

CSS border does not accept gradient values. The correct approach uses two background layers on the element itself: a padding-box layer for the solid fill (clipped inside the border area), and a border-box layer for the gradient (visible only in the border region). A border: 3px solid transparent creates the space for it to show through. Works perfectly on any border-radius.

.btn {
  background:
    #F8F5EC padding-box,             /* fill — clipped inside border */
    linear-gradient(90deg, #5E00FF, #CE6F31) border-box; /* gradient stroke */
  border: 3px solid transparent;        /* space for gradient to show */
  border-radius: 45px;
}
